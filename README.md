# WDev - Clone Any WordPress Site Instantly

> **Pronounce it as "Dub-Dev"**

WDev is the modern staging environment tool that lets you **clone any WordPress site with a single command**.  Perfect for agencies, developers, and teams who need instant staging environments with real production data.

```bash
wdev pull myblog.com
```
Your site is now available at http://myblog.test

## Why WDev?

- **🔄 One-Command Cloning** - Pull production sites to local with `wdev pull`
- **🎯 Perfect Staging** - Exact replicas with automatic URL rewriting  
- **⚡ Zero Configuration** - Docker-powered with intelligent automation


## Use Cases

- **🏢 Agency Development** - Clone client sites for maintenance and features
- **🐛 Bug Reproduction** - Debug with exact production data safely  
- **🧪 Feature Testing** - Test updates on production replicas
- **📦 Data Migration** - Move sites between hosts with simple URL rewriting
